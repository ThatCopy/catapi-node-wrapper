# A nodejs API wrapper for catAPI

### To install

```bash
npm i --save catapi
```

_or_

```bash
yarn add catapi
```

## Docs

##### You should find JSDoc in your IDE

```js
console.log(await random()) // Returns a random cat.
```

```js
console.log(await id(1)) // Search for cat by id.
// The id in the json is _id
```
